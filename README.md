Prediction Market Smart Contract

This repository contains a Prediction Market smart contract built on Solidity.

The contract allows users to create prediction markets, place bets, and claim rewards based on the outcome.

Features

Create prediction events (markets)

Users can place predictions on available outcomes

Funds are locked until the event is resolved

Admin resolves the event and distributes winnings automatically

Transparent, decentralized & verifiable on-chainprediction the market
